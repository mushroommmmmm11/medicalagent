### AI 讲解提供

AI 助手基于代码分析，详细讲解了以下 7 个处理阶段：

1. **前端上传** - Vue 选择图片，POST 到 /upload-report
2. **Java 后端处理** - 文件保存 + 异步 OCR 预热
3. **异步 OCR 识别** - Python 服务调用 Qwen/GPT-4o
4. **前端自动发送** - 上传成功后自动生成分析消息
5. **Agent 分析**：
   - RAG 知识库检索
   - 加载患者病历
   - 【新增】医学知识库检验结果分析
   - 构建 LLM 提示
   - 流式输出诊断
6. **前端显示** - 提取 META 元数据，显示确认按钮
7. **病历保存** - 用户确认后数据库记录

## 对话关键要点

### 系统设计理念

- **异步优先**：上传立即返回，OCR 后台进行
- **自动化流程**：上传成功自动发起分析，不需用户重复操作
- **流式体验**：分析结果逐词显示，提升用户感受
- **知识库驱动**：结合医学参考范围、RAG 文献、患者病历进行综合分析
- **可确认记录**：诊断结果由用户确认后才记录到病历（谨慎态度）

### 技术栈实现

- **前端**：Vue 3 流式 UI 更新
- **后端**：Java Spring Boot + WebClient（异步 HTTP）
- **AI 服务**：Python FastAPI + 异步处理
- **向量库**：FAISS 存储医学文献
- **缓存**：Redis（可选）优化性能
- **LLM**：DashScope (默认) / OpenAI / Anthropic
